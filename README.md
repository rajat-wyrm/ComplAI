# AI Compliance & Risk Copilot

RegTech AI system for analyzing legal and compliance documents.

## Features
- Document analysis (PDF/DOCX)
- Risk detection and scoring
- Actionable insights
- Conversational AI
- Dashboard analytics

## Tech Stack
- Backend: FastAPI, MongoDB, Redis, Celery
- Frontend: Next.js 14, TailwindCSS
- AI: DeepSeek API, RAG pipeline
- DevOps: Docker, Cloud Run
